# ☕ Coffee House - Modern Coffee Website

A responsive, modern, and visually appealing multi-page website for a coffee shop. Built using HTML, CSS, and JavaScript.

## 🚀 Demo
Check out the live site here: [https://harris8099.github.io/Demo-coffee-website/]

## ✨ Features
- **Responsive Design:** Optimized for desktop, tablet, and mobile devices.
- **Interactive Menu:** Browse different coffee blends, pastries, and snacks.
- **About Section:** Learn about our story and our beans.
- **Contact Form:** Functional contact section for customer inquiries.
- **Smooth Animations:** Integrated CSS animations for a premium feel.

## 🛠️ Technologies Used
- **HTML5:** For the structure of the website.
- **CSS3:** For styling, including Flexbox and Grid.
- **JavaScript:** For mobile navigation and interactive elements.
- **Google Fonts:** For typography.
- **Font Awesome:** For social media and UI icons.
